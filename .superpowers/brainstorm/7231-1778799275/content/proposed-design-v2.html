Does This Cover the Core?

The spec and this mockup cover:

  • Compact 160px track headers with gain/pan faders, mute/arm/print/remove buttons
  • Offwhite timeline with subtle grid lines, bar ruler, color-coded clips
  • Clip states: default, hover (lift), select (amber ring + glow), drag ghost
  • Resize handles on selected clips, split button on hover
  • Snap line + tooltip during drag/resize
  • Playhead with triangle head + grab handle + vertical line
  • Keyboard shortcuts (Space, R, Delete, Cmd+D, arrows)
  • Zoom (Ctrl+wheel / pinch)

What it does NOT cover (future pass): comping/takes, punch-in recording, MIDI piano roll, automation lanes, cross-fades.
