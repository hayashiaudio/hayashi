How Bandlab-Style Track Colors Work

  • Each track has a selectable color — not derived from source kind, user picks it
  • Track header background is tinted with a light version of the track color
  • Clip lane background is also tinted with the same light color for visual grouping
  • Clips themselves use a slightly more saturated version of the track color
  • Selected clip border is always amber/orange to stand out regardless of track color
  • Right-click track header opens a color swatch picker (shown above)
  • Default colors are assigned automatically when tracks are created, but user can change anytime
